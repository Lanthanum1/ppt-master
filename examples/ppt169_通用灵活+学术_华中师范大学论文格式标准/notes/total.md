# slide_01_cover

本页为封面页，展示华中师范大学研究生学位论文撰写格式标准的主题，适用于硕士和博士研究生。依据 GB/T 7713-1987 和 GB/T 7714-2005。

# slide_02_contents

本页为目录，共 9 个章节：论文基本结构、前置部分要求、摘要与关键词规范、正文撰写规范、图表公式格式、参考文献格式、附录与致谢、排版格式要点、格式检查清单。

# slide_03_structure

华中师范大学学位论文由十二大部分组成：封面、中英文扉页、原创性声明与授权书、中英文摘要、目录、正文、参考文献、附录（如有）、攻读期间发表论著、致谢。页码规则：封面和目录不编页码，摘要用罗马数字，正文起用阿拉伯数字。

# slide_04_front_matter

前置部分包括：封面（单面印刷）、中文扉页（双面）、英文扉页（双面）、原创性声明与授权书（单面，必须亲笔签名）、中文摘要（罗马数字页码）、英文摘要（罗马数字页码）。

# slide_05_abstract

中文摘要："摘要"三号黑体加粗居中，中间空两格；正文硕士小四号宋体/博士五号宋体；行距硕士多倍1.25/博士固定19磅；关键词3-8个，分号间隔。英文摘要："Abstract"三号Times New Roman加粗居中；非中文撰写的论文须附中文摘要，博士≥8000字，硕士≥3000字。

# slide_06_main_body

一级标题：三号黑体加粗居中，另起一页。二级标题：四号黑体加粗，首行缩进。三级标题：小四号黑体加粗，首行缩进。正文：硕士小四宋体/博士五号宋体，字母数字用Times New Roman，首行缩进两字符。注释采用脚注，每页重新编号。

# slide_07_figures_tables

图：编号图2.1格式，图题在图的下方居中；表：编号表2.1格式，表题在表的上方居中，推荐三线表；公式：另起一行居中，分章节编号，编号在右侧圆括号内，转行在运算符处。

# slide_08_references

采用顺序编码制，引用序号以上角标方括号形式出现。作者不超过三位全部列出，超过三位列前三位加"等"或"et al"。文献类型标识：[M]专著、[J]期刊、[D]学位论文、[C]会议录等。参考文献标题三号黑体加粗居中，另起一页。

# slide_09_appendix

附录可收录公式推导、实验数据、程序代码、攻读期发表论著（博士必须）、阅读型参考文献等，编号为附录A、B、C。致谢位于论文末尾，感谢导师、评阅人、资助单位、家人等，格式与正文相同。

# slide_10_layout

硕士左边距2.8cm，右边距3.0cm，正文小四号宋体，多倍行距1.25，页码居中；博士左右边距均2.9cm，正文五号宋体，固定行距19磅，页码外侧；上下边距相同（上3.9cm，下3.5cm），页眉1.5cm，页脚2.7cm。

# slide_11_checklist

提交前逐项核查：封面信息、签名声明、摘要格式、关键词数量、标题层级、图表编号、参考文献格式、页边距、字体字号、页码规则、印刷方式、电子版提交等12项。
